# A collection of C++ Apps / Libraries

- **MacroLand**: An extensible and customizable computing environment. Mostly written in C++ and uses Python for extending the framework. 
  
- **JSON Parser:** A portable, single header/cpp library which parses JSON text/file and serializes C++ data to JSON text. 
   
- **String to Number**: Converts a string to float/double using transition diagram.  
  
- **cpp manager**: A simple command-line tool to generate new cpp projects.  
  
- **Result class**: Rust inspired templated Result class to return *Ok* or *Err*.

- **Bisection**: A generic, type-safe bisection root finding method.
